##  1. 운영규칙 

 - 매주 1회의 만남을 가지도록한다.<br>
 - '파이토치 첫걸음' e-book 활용
 - 주차별 챕터 요약문 각자 올리기

   순서 : 상원, 예지, 문경, 정은, 유재, 지수, 수진, 남규
 - 발표자 이외에 팀원들은 발표를 하는 동안 무조건 질문 1가지 이상을 하도록 한다.
 - 발표가 끝나고 질의응답을 통해 서로 부족한 부분을 얘기하고 알아가도록 한다.
 - 이 폴더에 각자의 폴더를 만든 후, 그곳에 자신이 공부한 자료를 남기기로 한다.
